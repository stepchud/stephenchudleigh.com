Installation

Fortunately, this is pretty easy. You need priveleges to install software on your computer. If you already have Java version 1.4.2 or newer installed, follow the instructions below to Download & Play. Otherwise, you will need to go to the Java download page and get the latest version.

Click the Java image below and following their installation instructions.

Download & Play

Due to some security updates to Java, the easiest thing to do is download CBGame.jar and play it directly from your computer:

  1. Right click this link, choose "Save As..."
  2. Save it to your hard drive, e.g. your Desktop.
  3. Navigate to that location and double-click the file.
  4. The game should start but if not, try right-click and choose "Open".
  5. On a Mac you may get a warning about unidentified developer. That's me.
  6. Hold down the "Control" key, right-click the file and choose open. Then select "Open" at the confirmation prompt.

Don't worry! This game does not even connect to the internet, much less send your personal data anywhere. Weird, right?
Please contact me if you have problems starting the game or to report bugs.
