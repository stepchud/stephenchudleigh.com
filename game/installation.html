Installation

Fortunately, this is pretty easy. You may need to have priveleges to install software on your computer only if you do not have a J2SE runtime environment installed already. If you have the J2SE runtime with Java Web Start already installed, then you are ready to

start playing now.

Otherwise, you will need to go to the sun website and get the latest runtime environment by clicking the button below and following their installation instructions.

If you still have trouble with getting the link to launch the game after you install java from the Sun website, you might have more luck with launching it directly from your own computer.
Right click this link, choose "Save As..." and save it to a directory on your hard drive, e.g. C:\cbg\ Then, open a command prompt by clicking
Start Menu -> Run...
type "cmd" in the box
click Ok.
Assuming you saved the file to C:\cbg, type the following commands at the DOS prompt:
cd "C:\cbg"
java -jar CBGame.jar
 If you have a compatible version of java installed, this should be enough to start a game. Please contact me if you have problems starting the game or to report bugs.
